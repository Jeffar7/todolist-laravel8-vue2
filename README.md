# LARAVEL SETUP

1. Setup the the .env file

    DB_DATABASE=todolist

    DB_USERNAME= _yourusername_

    DB_PASSWORD= _yourpassword_

2. For Running the project use this command

    > "php artisan migrate"

    > "php artisan serve"

#

# VUE SETUP

1. For Running Vue use this command

    > "npm run hot"
